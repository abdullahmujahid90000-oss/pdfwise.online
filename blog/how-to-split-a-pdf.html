How to Split a PDF into Separate Pages or Sections

By PDFWise Team  ·  June 2026  ·  5 min read

Whether you need to extract a single page from a large report, share only specific chapters of an ebook, or break a 200-page document into smaller files, splitting a PDF is one of the most common tasks people do with PDF files. The good news: you don't need expensive software to do it. You can split any PDF for free, right in your browser.

This guide covers the different ways to split a PDF and when to use each approach.

When Would You Need to Split a PDF?

There are several common situations where splitting a PDF makes sense:

  • Email attachment limits: Many email providers limit attachments to 10–25 MB. Splitting a large PDF lets you send it in multiple smaller parts.
  • Sharing only relevant pages: You have a 50-page contract but only need to send pages 12–18 to a client. Splitting extracts just those pages.
  • Separating scanned documents: A batch scan creates one large PDF with multiple separate documents. Splitting divides them back into individual files.
  • Extracting individual pages: You want just one page — a certificate, a receipt, or a signature page — as its own standalone PDF.
  • Reducing file size: Removing pages you don't need is another way to shrink a PDF alongside compression.

The Three Ways to Split a PDF

1. Split into Individual Pages

This is the most thorough option — every page becomes its own separate PDF file. If you have a 10-page document, you end up with 10 individual files. This is useful when each page is a distinct document (forms, invoices, certificates) and you want to handle them separately.

2. Extract Specific Pages

You choose exactly which pages to keep, and the tool creates a new PDF containing only those pages. For example, from a 40-page report you might extract pages 5, 10, and 15–20. The remaining pages are discarded from the new file (your original is never modified).

3. Split at a Specific Page

You choose a page number, and the tool splits the document into two PDFs at that point. Pages 1 to N go into the first file, pages N+1 to the end go into the second. This is ideal for splitting a document into clear halves or chapters.

Tips and Things to Know

  • Your original file is never modified. The split tool always works on a copy — your uploaded PDF stays exactly as it was.
  • Password-protected PDFs need to be unlocked first. Use the Unlock PDF tool before splitting.
  • File size is proportional to page count. If you extract 10 pages from a 100-page PDF, the resulting file will be roughly 10% of the original size (exact results depend on which pages contain large images).
  • Page numbering in the PDF tool refers to the actual PDF pages (1, 2, 3...), not any printed page numbers that might appear in the document's content.
